# Carte d'apprentissage AWS SAA

## Semaine 1 - Fondation

- Revoir IAM, S3, CloudFront, API Gateway, Lambda, DynamoDB.
- Lire l'architecture du projet.
- Faire un premier `terraform plan`.

Livrable portfolio : diagramme d'architecture + README clair.

## Semaine 2 - Deploiement minimal

- Deployer l'API et DynamoDB.
- Tester les endpoints.
- Deployer le frontend statique.

Livrable portfolio : URL de demo + captures.

## Semaine 3 - Securite et operations

- Ajouter budget, tags, logs et alarmes.
- Documenter les decisions d'architecture.
- Verifier les permissions IAM.

Livrable portfolio : section "Security and Operations".

## Semaine 4 - Extension

Choisir une extension :

- Cognito pour l'authentification ;
- GitHub Actions pour CI/CD ;
- EventBridge + SNS pour notifications ;
- WAF pour protection edge.

Livrable portfolio : mini article technique expliquant le compromis choisi.
